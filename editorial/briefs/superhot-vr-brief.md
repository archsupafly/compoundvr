# SUPERHOT VR Brief

## Game Context
- **Title:** SUPERHOT VR
- **Release:** December 2016 (Quest: 2019)
- **Developer:** SUPERHOT Team
- **Genre:** First-person shooter, puzzle
- **Core gameplay:** Time moves only when you move — strategic gunfight puzzle where you dodge bullets in slow motion
- **Known for:** Innovative time mechanic, minimalist red-on-white aesthetic, satisfying combat flow

## Route Type
**Full VR Game (Separate from Flat)** — This is NOT the same game as flat SUPERHOT. The VR version has different levels, different mechanics, different campaign. It's a separate purchase.
