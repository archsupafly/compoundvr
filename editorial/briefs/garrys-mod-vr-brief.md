## Game-Specific Context

- Garry's Mod (2006) — physics sandbox built on Source engine
- Massive user-generated content ecosystem (Steam Workshop addons, game modes)
- VRMod created by Catse, first released March 2019
- Steam Workshop addon — requires module installation plus workshop subscription
- Supports motion controllers, full body tracking (Vive trackers), lip and eye tracking
- Works with all SteamVR headsets
- "Experimental" label suggests ongoing development, some rough edges
- Single player and multiplayer support
