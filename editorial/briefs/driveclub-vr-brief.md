## Route Type Context

This is an **Official Standalone VR Version** — a VR-only version of an existing flat game. DRIVECLUB (2014) was a PS4 racing game. DRIVECLUB VR (2016) is a separate VR-only release, not a mode you toggle in the base game. This is different from "Official Hybrid" where the same game has VR support built in.

---

## Game-Specific Context

- DRIVECLUB VR released October 2016 as PSVR launch title
- Separate purchase from base DRIVECLUB (not DLC, not a mode)
- Based on DRIVECLUB (2014) — PS4 racing game by Evolution Studios
- Developer: Evolution Studios (same studio as flat game)
- Server shutdown: August 2019 for base game, features limited after
- Graphics significantly downgraded from flat version (PS2-era visuals criticism)
- Same tracks, same cars, same campaign structure as flat game
- VR-only experience — cannot play without headset
