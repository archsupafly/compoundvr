# Euro Truck Simulator 2 VR Brief

## Game Context
- **Title:** Euro Truck Simulator 2 (ETS2)
- **Release:** 2012 (ongoing development)
- **Developer:** SCS Software
- **Genre:** Simulation, driving
- **Core gameplay:** Drive trucks across Europe, deliver cargo, build a trucking business
- **Known for:** Relaxing gameplay, meticulous attention to truck physics and cabin detail, massive map expansion

## Route Type
**Official Standalone VR Version** — Native VR support released by developer

## Platforms
- PCVR (SteamVR)
- Quest via Quest Link / Virtual Desktop (PCVR streaming)

## Research Focus

- Comfort options (vignette, teleport, snap turn)
- Performance requirements — this is a demanding game
- Multiplayer/session compatibility with flat players
- DLC and mod support in VR
- Known issues or limitations
