# Doom VR Brief

## Game Context
- **Title:** Doom (1993)
- **Release:** December 1993
- **Developer:** id Software
- **Genre:** First-person shooter
- **Core gameplay:** Classic demon-slaying FPS through Mars bases and Hell itself. Fast-paced run-and-gun combat, key hunting, secret walls, iconic weapons (shotgun, chaingun, BFG)
- **Known for:** Popularized the FPS genre, invented deathmatch, massive modding community, source code released in 1997 enabling countless ports

## Route Type
**Framework-Based / Source Port VR** — Not an official VR release. VR support comes from community source ports: QuestZDoom (Quest standalone), various PC VR source ports (gzdoom-VR, etc.)

## Platforms
- Quest (standalone via QuestZDoom)
- PCVR (source ports)

## Research Focus
- What VR source ports exist? (QuestZDoom, gzdoom-VR, others?)
- How complete is the VR implementation? (motion controls, comfort, weapon handling)
- Does it support Doom 1, Doom 2, Final Doom, or all of them?
- QuestZDoom standalone vs PC VR source ports — which is better?
- Can you use custom WADs? (important for Doom community)
- Performance on Quest vs PC
- Comfort options (snap turn, vignette, movement options)
- Known issues or limitations
