## Route Type Context

Framework-Based: 3dSen VR is a real-time 2D-to-3D voxel transformation engine that converts NES games into stereoscopic VR environments. Not an emulator — a dedicated VR application that loads NES ROMs and renders them as explorable 3D scenes.

---

## Game-Specific Context

Metroid (1986) is a foundational Metroidvania — Nintendo's exploration-focused action platformer that defined the genre. Samus Aran's debut mission on Planet Zebes.

3dSen VR supports Metroid in its library of playable NES games. The VR experience converts the 2D side-scroller into a 3D voxel diorama, allowing depth perception and a new visual interpretation of the classic world.

This is not a native VR port or community mod of the game itself — it's a separate application (3dSen VR) that provides a VR window into the NES classic.
