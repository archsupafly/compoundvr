# VR Modding Flat Games

**Canonical Definition:** Build a business around flat-to-VR mods — starting as an authority website that catalogs, reviews, and evaluates mods, then evolving into a project that can eventually release original VR mods.

**Status:** Structured experiment — initial team and operating model now defined.

## Current Thesis
This project now has a two-stage path:

1. **Editorial / discovery layer**
   - catalog flat-to-VR mods
   - review and compare them honestly
   - build authority around install friction, compatibility, comfort, and playability

2. **Production / release layer**
   - identify viable mod opportunities
   - scope and coordinate technical work
   - eventually publish original VR mods

## Open Questions
