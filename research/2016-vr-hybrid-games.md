# 2016 VR Games: Hybrid and Flat-Based Properties

**Research Date:** 2026-03-21
**Purpose:** Identify VR games launched in 2016 that were either:
1. Hybrid (VR + non-VR in same package)
2. Based on existing flat game properties (VR version of flat game)

---

## PSVR Launch (October 13, 2016)

### Hybrid (VR + Non-VR)
| Game | Original Flat | Notes |
|------|---------------|-------|
| **Rez Infinite** | Yes (2001/Remaster) | VR-compatible, entire game playable flat or VR. Already covered on site. |
| **Thumper** | No (launched same day) | "Rhythm violence" game, VR-compatible, playable flat. VR optional from launch. |
| **Resident Evil 7: Biohazard** | Yes (Jan 2017) | Horror game, VR mode exclusive to PSVR for 1 year. Not hybrid — VR mode is separate. |

## Key Findings

### True Hybrids (Same Game, VR Optional)
- **Elite Dangerous** — Same game, VR mode added, toggle between flat and VR
- **Project CARS** — Same game, VR mode added, toggle between flat and VR
- **Rez Infinite** — Same game, VR optional from launch
- **Thumper** — Same game, VR optional from launch
- **Minecraft VR** — Same game, VR optional
- **The Talos Principle VR** — Same game, VR version
- **Defense Grid 2 Enhanced VR Edition** — Same game, VR-enhanced

### VR Versions of Flat Games (Separate Purchase/Release)
- **DRIVECLUB VR** — VR-only version of DRIVECLUB, separate purchase
- **The Vanishing of Ethan Carter** — VR version of adventure game
- **AirMech: Command** — VR version of strategy game
- **Serious Sam 3 VR: BFE** — VR version of FPS
- **SUPERHOT VR** — VR version with different levels (not same campaign)
- **Pinball FX2 VR** — VR version of pinball game

### VR-First, Later Added Non-VR
- **EVE: Valkyrie** — VR-only at launch (March 2016), added non-VR in September 2017
