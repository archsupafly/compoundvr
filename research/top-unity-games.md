# Top 100 Unity Engine Games

Master list of notable Unity engine games, organized by year. Used to identify potential VRGIN candidates and track which Unity games have VR paths.

**Purpose:** Identify Unity games that could potentially work with VRGIN injection framework (best with Unity 5.x - 5.6 games).

---

## What is VRGIN?

VRGIN is a VR injection framework for Unity games created by Eusth. It enables VR support for Unity games that don't natively support it. Works best with Unity 5.x games (5.0-5.6). Newer Unity versions may break injection.

**Known VRGIN-Compatible Games:**
- Firewatch (2016)
- Dear Esther: Landmark Edition (2015)
- Ghost of a Tale (2016)
- Layers of Fear (2016)
- DARQ (2019)
- Yooka-Laylee (2017)
- Kona (2017)
- Night in the Woods (2017)
- Broforce (2015)
- Valley (2016)
