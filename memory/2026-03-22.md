### Maya Workflow Discovery
**Critical Pattern Learned:** Richard was over-directing Maya on hero image generation.

**Wrong approach (current session):**
- Giving narrative context ("beloved 2007 multiplayer shooter visible through a VR headset...")
- Giving specific visual directions ("VR headset concept", "faded character silhouettes")
- Adding process instructions ("Use gemini-image-simple", "Reference the brand guide")

**Correct approach (earlier sessions - Dishonored, Rez Infinite, etc.):**
- Minimal framing only: "Generate a hero image for the game [GAME NAME]. Save to: [path]"
- Maya reads brand guide herself
- Maya crafts the visual prompt based on brand aesthetics + game identity
- Maya decides composition, style, and approach

**Why it matters:** Over-directing constrains Maya's creative interpretation. The brand guide exists so Maya can reference it independently. Richard's job is coordination, not art direction.

**Documentation:** Pattern documented in MEMORY.md Agent Workflow section
